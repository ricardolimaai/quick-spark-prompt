# Quick Spark Prompt

A community-driven platform for discovering and sharing AI prompts, image generation prompts, and automation workflows.

## Features

- Browse prompts by category: text, image generation, and automation
- Masonry card layout with upvotes, views, and comments
- Search and filter by content type
- Clean, dark UI optimized for content discovery

## Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + TypeScript + Vite |
| UI Components | shadcn/ui + Tailwind CSS |
| State | TanStack Query |
| Routing | React Router |

## Getting Started

```bash
npm install
npm run dev
```

## Use Case

Built as a tool to organize and share prompt engineering work. Useful for AI practitioners who want to catalog and discover effective prompts across different models and use cases.
